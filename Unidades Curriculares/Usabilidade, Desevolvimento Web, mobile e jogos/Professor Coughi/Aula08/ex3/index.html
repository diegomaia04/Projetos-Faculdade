Ola Mundo

nesta pagina vamos fazer alteração do texto da tag h1 quando o o usuario clicr no botao. para fazer isso temos que usar o comando onClick

onClick criara um evento de clic no botao. este evento estara atrelado a uma função

        
         <script>
         let h1 = document.querySelector('h1')
         const btn = document.querySelector('button')
 
         btn.onclick = function(){
             h1.innerHTML ="Adeus Mundo"
         }   
         </script>
